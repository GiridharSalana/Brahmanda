# Sanatan Cosmos - Interactive 3D Visualization

An immersive, interactive website showcasing the hierarchical structure of Sanatan Dharma (Hinduism) through modern 3D visualization and beautiful UI design.

## 🕉️ Features

### 3D Visualization
- **Interactive 3D spheres** representing each level of the cosmic hierarchy
- **Dynamic connections** between levels with curved lines
- **Particle system** creating a cosmic atmosphere
- **Glow effects** and shaders for enhanced visual appeal
- **Parallax mouse tracking** for immersive experience

### Modern UI/UX
- **Gradient color schemes** with gold, purple, and cosmic colors
- **Smooth animations** and transitions
- **Scroll-based interactions** that sync with 3D view
- **Responsive design** for all devices
- **Touch support** for mobile devices

### Content Structure
1. **Parama Brahman** - Ultimate Reality
2. **Cosmic Realms** - The 14 Lokas
3. **Cosmic Time Cycles** - Yugas and Kalpas
4. **Avatars & Divine Incarnations** - Vishnu, Shiva, Shakti manifestations
5. **Deities & Cosmic Forces** - 33 Koti Devas
6. **Itihasa & Epics** - Ramayana, Mahabharata, Puranas
7. **Human Realm** - Varna, Ashrama, Purusharthas
8. **Karma & Reincarnation** - Cycle of rebirth
9. **Shruti & Smriti** - Sacred scriptures
10. **Moksha** - Liberation and paths to enlightenment

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or build process required!

### Running the Site

1. **Clone or download this repository**
   ```bash
   cd SanatanCosmos
   ```

2. **Open with a local server** (recommended)
   
   Using Python:
   ```bash
   python -m http.server 8000
   ```
   
   Using Node.js:
   ```bash
   npx serve
   ```
   
   Or use VS Code's Live Server extension.

3. **Or simply open `index.html`** directly in your browser
   ```bash
   open index.html
   ```

## 🎮 Controls

### Mouse/Keyboard
- **Scroll** - Navigate through hierarchy levels
- **Mouse Move** - Parallax camera movement
- **Arrow Up/Down** - Zoom in/out
- **R** - Reset camera view
- **Space** - Toggle rotation
- **Escape** - Close info panel

### UI Controls
- **+ Button** - Zoom in
- **− Button** - Zoom out
- **⟲ Button** - Reset view
- **⟳ Button** - Toggle auto-rotation

### Navigation
- **3D View** - Focus on 3D visualization
- **Hierarchy** - View detailed text content
- **Timeline** - Jump to Time Cycles section
- **About** - Learn more about the project

## 🛠️ Technologies Used

- **Three.js** - 3D graphics library (r128)
- **Vanilla JavaScript** - No frameworks, pure performance
- **CSS3** - Modern animations and gradients
- **Google Fonts** - Cinzel, Inter, Noto Sans Devanagari
- **WebGL** - Hardware-accelerated 3D rendering

## 🎨 Design Philosophy

### Visual Hierarchy
- Top level (Brahman) uses gold gradients representing ultimate divinity
- Middle levels use purple/blue representing cosmic realms
- Bottom level (Moksha) returns to gold, symbolizing reunion

### Color Palette
- **Gold (#FFD700)** - Divine, ultimate reality
- **Purple (#8B5CF6)** - Cosmic, mystical
- **Orange (#FF6B35)** - Energy, transformation
- **Blue (#3B82F6)** - Peace, preservation

### Typography
- **Cinzel** - Elegant serif for headings
- **Inter** - Clean sans-serif for body
- **Noto Sans Devanagari** - Authentic Sanskrit symbols
